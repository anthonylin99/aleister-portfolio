OVERVIEW
--------
Complete frontend design transformation inspired by Stripe's premium aesthetic.
This update replaces the previous Yankee Candle Lavender theme with a vibrant,
flowing gradient system featuring purple → pink → coral color transitions.

Design Influences:
- Stripe's flowing gradient backgrounds and "stream to the right" animations
- Robinhood's clean data presentation and micro-interactions
- Luma's elegant glassmorphism and card layouts

Key Design Principles Applied:
- Premium, alive UI that inspires confidence
- Thoughtful animations that guide the eye
- Color gradients that flow naturally
- Every element designed with intention

New CSS Components Created:
---------------------------

1. .stripe-gradient-bg
   - Fixed position flowing gradient background
   - Uses conic-gradient with all Stripe colors
   - 20s rotation animation for subtle movement
   - Blurred for soft, ambient effect

2. .stream-right
   - Horizontal "stream to the right" effect
   - Linear gradient that flows continuously
   - 8s animation cycle
   - Used for subtle motion on hero sections

3. .hero-gradient
   - Three floating gradient orbs
   - gradient-orb-1: Purple (800x800px, top-right)
   - gradient-orb-2: Fuchsia/pink (600x600px, center-right)
   - gradient-orb-3: Coral/orange (500x500px, bottom-right)
   - Float animation with 15s cycle

4. .gradient-card
   - Premium card with gradient border effect
   - Border uses purple → fuchsia → coral gradient
   - Opacity animates from 0.5 to 1.0 on hover
   - 4px translateY + shadow on hover

5. .bento-card
   - 5 gradient variants (.gradient-1 through .gradient-5)
   - Each with unique color combination
   - Scale + translateY animation on hover
   - Used for feature sections

6. .stat-card
   - Top gradient accent line
   - Gradient background (subtle)
   - Icon container with gradient fill
   - Hover lifts card 2px

7. .holding-card
   - Bottom accent bar (reveals on hover)
   - .positive class: green gradient bar
   - .negative class: red gradient bar
   - Gradient border appears on hover

8. .btn-primary / .btn-secondary
   - Stripe-style buttons
   - Primary: gradient background with glow shadow
   - Secondary: transparent with gradient border
   - Scale animation on click (0.98)

================================================================================
DESIGN PHILOSOPHY NOTES
================================================================================

Per bible.md principles applied:

1. "Leave the campsite cleaner" - Removed old lavender theme entirely,
   replaced with cohesive Stripe-inspired system

2. "Fail loudly" - All TypeScript errors fixed, no any types

3. "Predictable patterns" - Consistent gradient naming (gradient-1 through 5)

4. "Self-documenting" - JSDoc comments added to all major components

5. "Long-term stewardship" - CSS variables allow easy future modifications

Color Psychology Applied:
- Purple (#635bff): Trust, sophistication, innovation
- Pink/Fuchsia: Energy, excitement
- Coral/Orange: Warmth, optimism
- Used together: Creates sense of premium, forward-thinking fintech

Animation Philosophy:
- All animations serve a purpose (guide attention, show state)
- Durations kept under 0.5s for responsiveness
- Easing curves chosen for natural feel
- No animation is purely decorative
